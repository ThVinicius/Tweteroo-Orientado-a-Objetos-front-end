## Rodar localmente

Clone o projeto

```bash
  git clone https://github.com/ThVinicius/Tweteroo-Orientado-a-Objetos-front-end.git
```

Vá para o diretório do projeto

```bash
  cd Tweteroo-Orientado-a-Objetos-front-end
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
